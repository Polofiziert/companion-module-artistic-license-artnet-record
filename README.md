# Artnet Librairy "Artnet-Node"

## ArtPollReply packet
*art-node/lib/artnet_server - discover*

The discover function sends a ArtPoll Packet over the UDP socket on the given Address on the fixed port of 6454(dec) 0x1936 (hex)

When it gets an response it is in form of an ArtPollReply Packet.
